# CJR Performance App - Publishing Guide

## Overview
This guide provides step-by-step instructions for publishing the CJR Performance App to both Google Play Store (Android) and Apple App Store (iOS).

## Prerequisites

### Required Accounts
- **Google Play Developer Account** ($25 one-time fee)
- - **Apple Developer Program** ($99/year)
  - - **EAS Account** (Free - for building with Expo)
    - - GitHub account (already have)
     
      - ### Required Tools
      - ```bash
        npm install -g eas-cli
        npm install -g expo-cli
        ```

        ### Project Setup
        1. Ensure `app.json` is configured correctly
        2. 2. Ensure `eas.json` is in place
           3. 3. Update version numbers before each release
              4. 4. Create required assets (icons, screenshots)
                
                 5. ---
                
                 6. ## PART 1: Setup EAS (Expo Application Services)
                
                 7. ### Step 1: Initialize EAS Project
                 8. ```bash
                    cd cjr-mobile
                    eas project:create
                    ```
                    This creates your EAS project and saves the projectId to `app.json`.

                    ### Step 2: Update app.json
                    Ensure the projectId is in `app.json`:
                    ```json
                    "extra": {
                      "eas": {
                        "projectId": "your-project-id-here"
                      }
                    }
                    ```

                    ### Step 3: Login to EAS
                    ```bash
                    eas login
                    ```
                    Enter your credentials (create account at https://expo.dev if needed).

                    ---

                    ## PART 2: Android Publishing (Google Play)

                    ### Step 1: Create Google Play Developer Account
                    1. Visit https://play.google.com/console
                    2. 2. Pay $25 registration fee
                       3. 3. Complete merchant setup
                          4. 4. Add payment information
                            
                             5. ### Step 2: Create App Listing
                             6. 1. In Google Play Console: Click "Create app"
                                2. 2. Fill in:
                                   3.    - App name: "CJR Performance"
                                         -    - Default language: English
                                              -    - App category: Automotive
                                                   -    - Free or Paid: Select appropriate option
                                                        - 3. Accept policies and create
                                                         
                                                          4. ### Step 3: Build APK/AAB with EAS
                                                          5. ```bash
                                                             # Build for production
                                                             eas build --platform android --type app-bundle

                                                             # Or if you prefer APK
                                                             eas build --platform android --type apk
                                                             ```

                                                             This creates a signed build ready for Google Play.

                                                             ### Step 4: Upload Build
                                                             1. In Google Play Console, go to "Internal testing"
                                                             2. 2. Upload the `.aab` file from your EAS build
                                                                3. 3. Review and publish to internal testing first
                                                                  
                                                                   4. ### Step 5: Add App Details
                                                                   5. 1. **App description**: ~500 characters
                                                                      2.    - Describe key features
                                                                            -    - Highlight performance tracking capabilities
                                                                                 -    - Mention real-time sync
                                                                                  
                                                                                      - 2. **Screenshots**: Create 2-5 screenshots
                                                                                        3.    - Size: 1080x1920 or 1440x2560
                                                                                              -    - Show key features
                                                                                                   -    - Include callouts if possible
                                                                                                    
                                                                                                        - 3. **App icon**: 512x512 PNG
                                                                                                          4.    - Must be exact dimensions
                                                                                                                -    - High quality, professional design
                                                                                                                 
                                                                                                                     - 4. **Feature graphic**: 1024x500 (optional but recommended)
                                                                                                                      
                                                                                                                       5. ### Step 6: Content Rating
                                                                                                                       6. 1. Click "App content"
                                                                                                                          2. 2. Fill out questionnaire
                                                                                                                             3. 3. Submit for rating
                                                                                                                               
                                                                                                                                4. ### Step 7: Set Pricing & Distribution
                                                                                                                                5. 1. Select "Free" or "Paid"
                                                                                                                                   2. 2. Choose countries to distribute
                                                                                                                                      3. 3. Set pricing (if paid)
                                                                                                                                        
                                                                                                                                         4. ### Step 8: Review & Publish
                                                                                                                                         5. 1. Review all information
                                                                                                                                            2. 2. Click "Review" then "Publish to Production"
                                                                                                                                               3. 3. Google Play reviews within 2-4 hours
                                                                                                                                                 
                                                                                                                                                  4. ### Helpful Resources
                                                                                                                                                  5. - https://play.google.com/console/about/guides/releasewithconfidence/
                                                                                                                                                     - - https://developer.android.com/distribute/best-practices/launch
                                                                                                                                                      
                                                                                                                                                       - ---
                                                                                                                                                       
                                                                                                                                                       ## PART 3: iOS Publishing (App Store)
                                                                                                                                                       
                                                                                                                                                       ### Step 1: Enroll in Apple Developer Program
                                                                                                                                                       1. Visit https://developer.apple.com
                                                                                                                                                       2. 2. Sign in with Apple ID
                                                                                                                                                          3. 3. Click "Membership" and enroll
                                                                                                                                                             4. 4. Pay $99/year
                                                                                                                                                                5. 5. Complete identity verification (takes 24-48 hours)
                                                                                                                                                                  
                                                                                                                                                                   6. ### Step 2: Create App ID
                                                                                                                                                                   7. 1. Go to https://appstoreconnect.apple.com
                                                                                                                                                                      2. 2. "Users and Access" → Set up your account
                                                                                                                                                                         3. 3. Go to "Certificates, Identifiers & Profiles"
                                                                                                                                                                            4. 4. Create App ID:
                                                                                                                                                                               5.    - Bundle ID: `com.42speedsports.cjr`
                                                                                                                                                                                     -    - App name: "CJR Performance"
                                                                                                                                                                                      
                                                                                                                                                                                          - ### Step 3: Configure EAS for iOS
                                                                                                                                                                                          - Update `eas.json`:
                                                                                                                                                                                          - ```json
                                                                                                                                                                                            "ios": {
                                                                                                                                                                                              "production": {
                                                                                                                                                                                                "autoIncrement": true
                                                                                                                                                                                              }
                                                                                                                                                                                            }
                                                                                                                                                                                            ```
                                                                                                                                                                                            
                                                                                                                                                                                            ### Step 4: Build IPA with EAS
                                                                                                                                                                                            ```bash
                                                                                                                                                                                            # Build for production
                                                                                                                                                                                            eas build --platform ios --type app-store
                                                                                                                                                                                            ```
                                                                                                                                                                                            
                                                                                                                                                                                            EAS automatically handles signing and certificate management.
                                                                                                                                                                                            
                                                                                                                                                                                            ### Step 5: Create App Store Listing
                                                                                                                                                                                            1. In App Store Connect: Click "Apps"
                                                                                                                                                                                            2. 2. Click "+" → "New App"
                                                                                                                                                                                               3. 3. Select platform: iOS
                                                                                                                                                                                                  4. 4. Fill in:
                                                                                                                                                                                                     5.    - App name: "CJR Performance"
                                                                                                                                                                                                           -    - Primary language: English
                                                                                                                                                                                                                -    - Bundle ID: `com.42speedsports.cjr`
                                                                                                                                                                                                                     -    - SKU: Create unique identifier
                                                                                                                                                                                                                      
                                                                                                                                                                                                                          - ### Step 6: Add App Information
                                                                                                                                                                                                                          - 1. **Subtitle**: Brief description (30 chars max)
                                                                                                                                                                                                                            2. 2. **Description**: ~4000 characters max
                                                                                                                                                                                                                               3.    - Key features
                                                                                                                                                                                                                                     -    - Use cases
                                                                                                                                                                                                                                          -    - Technical benefits
                                                                                                                                                                                                                                           
                                                                                                                                                                                                                                               - 3. **Keywords**: Comma-separated (100 chars total)
                                                                                                                                                                                                                                                 4.    - Example: "vehicle,performance,tracking,diagnostics,car"
                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                       - 4. **Privacy Policy URL**: Required
                                                                                                                                                                                                                                                         5.    - Host on your website
                                                                                                                                                                                                                                                               -    - Link to `PRIVACY_POLICY.md`
                                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                    - 5. **Support URL**: Required
                                                                                                                                                                                                                                                                      6.    - Email or support page
                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                            - ### Step 7: Add Screenshots
                                                                                                                                                                                                                                                                            - 1. For each device type (iPhone, iPad if applicable):
                                                                                                                                                                                                                                                                              2.    - Create 2-5 high-quality screenshots
                                                                                                                                                                                                                                                                                    -    - File format: PNG or JPG
                                                                                                                                                                                                                                                                                         -    - Minimum size: 1170x2532 (iPhone)
                                                                                                                                                                                                                                                                                              -    - Maximum size: 1242x2688
                                                                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                                                   - 2. Add screenshot descriptions for context
                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                     3. ### Step 8: Add App Icon
                                                                                                                                                                                                                                                                                                     4. 1. Upload app icon: 1024x1024 PNG
                                                                                                                                                                                                                                                                                                        2. 2. No alpha transparency
                                                                                                                                                                                                                                                                                                           3. 3. Professional, high-quality design
                                                                                                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                                                                              4. ### Step 9: Set App Category
                                                                                                                                                                                                                                                                                                              5. - Primary: "Automotive"
                                                                                                                                                                                                                                                                                                                 - - Content rating: Fill out questionnaire
                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                   - ### Step 10: Submit Build
                                                                                                                                                                                                                                                                                                                   - 1. In "TestFlight" tab:
                                                                                                                                                                                                                                                                                                                     2.    - Upload your iOS build from EAS
                                                                                                                                                                                                                                                                                                                           -    - Add test notes
                                                                                                                                                                                                                                                                                                                                -    - Invite testers (optional)
                                                                                                                                                                                                                                                                                                                                 
                                                                                                                                                                                                                                                                                                                                     - 2. After internal testing, go to "Releases"
                                                                                                                                                                                                                                                                                                                                       3. 3. Select build for App Store
                                                                                                                                                                                                                                                                                                                                          4. 4. Add release notes
                                                                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                                                                             5. ### Step 11: Submit for Review
                                                                                                                                                                                                                                                                                                                                             6. 1. Click "Submit for Review"
                                                                                                                                                                                                                                                                                                                                                2. 2. Answer review information questions
                                                                                                                                                                                                                                                                                                                                                   3. 3. Confirm all details
                                                                                                                                                                                                                                                                                                                                                      4. 4. Submit
                                                                                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                                                                                         5. Apple reviews within 24-48 hours (sometimes longer).
                                                                                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                                                                                         6. ### Step 12: Release
                                                                                                                                                                                                                                                                                                                                                         7. 1. Once approved, choose release strategy:
                                                                                                                                                                                                                                                                                                                                                            2.    - "Automatic release" - Release immediately
                                                                                                                                                                                                                                                                                                                                                                  -    - "Manual release" - Release on your schedule
                                                                                                                                                                                                                                                                                                                                                                       - 2. Click "Release"
                                                                                                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                                                                                                         3. ### Helpful Resources
                                                                                                                                                                                                                                                                                                                                                                         4. - https://developer.apple.com/app-store/submissions/
                                                                                                                                                                                                                                                                                                                                                                            - - https://developer.apple.com/app-store/review/guidelines/
                                                                                                                                                                                                                                                                                                                                                                              - - https://developer.apple.com/design/

                                                                                                                                                                                                                                                                                                                                                                                ## PART 4: Updates and Maintenance
                                                                                                                                                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                                                                                                                                ### For New Versions
                                                                                                                                                                                                                                                                                                                                                                                1. Update version in `app.json`:
                                                                                                                                                                                                                                                                                                                                                                                2.    ```json
                                                                                                                                                                                                                                                                                                                                                                                         "version": "1.1.0"
                                                                                                                                                                                                                                                                                                                                                                                         ```
                                                                                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                                                                                      2. Increment build numbers:
                                                                                                                                                                                                                                                                                                                                                                                      3.    - iOS: `buildNumber` in `app.json`
                                                                                                                                                                                                                                                                                                                                                                                            -    - Android: `versionCode` in `app.json`
                                                                                                                                                                                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                                                                                                                                                                 - 3. Create new builds:
                                                                                                                                                                                                                                                                                                                                                                                                   4.    ```bash
                                                                                                                                                                                                                                                                                                                                                                                                            eas build --platform android
                                                                                                                                                                                                                                                                                                                                                                                                            eas build --platform ios
                                                                                                                                                                                                                                                                                                                                                                                                            ```
                                                                                                                                                                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                                                                                                                         4. Submit to respective stores
                                                                                                                                                                                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                                                                                                                                                         5. ### Version Numbering
                                                                                                                                                                                                                                                                                                                                                                                                         6. Use semantic versioning: MAJOR.MINOR.PATCH
                                                                                                                                                                                                                                                                                                                                                                                                         7. - 1.0.0 - Initial release
                                                                                                                                                                                                                                                                                                                                                                                                            - - 1.0.1 - Bug fixes only
                                                                                                                                                                                                                                                                                                                                                                                                              - - 1.1.0 - New features
                                                                                                                                                                                                                                                                                                                                                                                                                - - 2.0.0 - Major changes

                                                                                                                                                                                                                                                                                                                                                                                                                  ```
                                                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                                                  ### Certificate Issues (iOS)
                                                                                                                                                                                                                                                                                                                                                                                                                  ```bash
                                                                                                                                                                                                                                                                                                                                                                                                                  # Let EAS manage certificates automatically
                                                                                                                                                                                                                                                                                                                                                                                                                  # Or revoke and recreate in Apple Developer account
                                                                                                                                                                                                                                                                                                                                                                                                                  ```
                                                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                                                  ### Submission Rejected
                                                                                                                                                                                                                                                                                                                                                                                                                  - Review rejection reason carefully
                                                                                                                                                                                                                                                                                                                                                                                                                  - - Fix issues
                                                                                                                                                                                                                                                                                                                                                                                                                    - - Resubmit with release notes explaining fixes
